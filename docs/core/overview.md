# SWPR Protocol — Core Overview

This section introduces the fundamental concepts behind the SWPR Dust Recovery Protocol.

## Purpose
SWPR’s mission is simple:
Turn forgotten, abandoned, or trapped wallet dust into usable value.

## Why Dust Exists
- Swap decimals  
- Bridge leftovers  
- Gas fragment residues  
- Airdrop micros  
- Failed transaction crumbs  

## Core Idea
Aggregate micro-value across chains → convert into a unified tokenized output (SWPR) → return usable value to holders.
