# Guides Overview

This section provides simple, step-by-step explanations of how to use the SWPR Protocol.

Guides are written for:
- new users  
- developers  
- wallet integrators  
- community contributors  

## Types of Guides

### 1. User Guides
Explain how a normal wallet user interacts with the protocol:
- scanning for dust  
- recovering value  
- understanding SWPR output  

### 2. Technical Guides
Explain how developers and integrators connect to the system:
- API basics  
- endpoints  
- scan triggers  
- batch verification  

### 3. Example Scenarios
Realistic walkthroughs showing:
- what happens during a recovery  
- how dust becomes SWPR  
- what logs look like  
- how the user sees the results
