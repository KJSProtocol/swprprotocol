# SWPR â€” Dust Recovery Value Protocol
### Turn Dust Into Value.

Welcome to the official documentation hub for the **SWPR Dust Recovery Protocol**.

SWPR is a protocol layer for:
- Detecting dust across wallets and chains  
- Sweeping tiny, stranded balances  
- Converting recovered value into SWPR or other assets  
- Burning and routing value according to protocol tokenomics
