# Glossary

This glossary defines key terms used throughout the SWPR Protocol documentation.

---

## Dust
Tiny, unusable token fragments that cannot be transacted, swapped, or bridged efficiently.

## Batch
A grouped set of dust fragments processed together for conversion.

## Conversion Engine
The protocol component that transforms dust batches into SWPR value.

## Recovery
The process of turning dust into usable SWPR tokens.

## Amplification
Optional boosts or multipliers applied during certain rollout models.

## Fragment
An individual dust amount detected during a scan.

## Chain
A supported blockchain network (e.g., Ethereum, Solana, Cosmos).

## Token Family
A category of assets such as ERC-20, SPL, Cosmos-SDK tokens, etc.

## SWPR
The unified token output representing recovered value from dust conversions.
